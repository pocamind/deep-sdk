# deep-sdk
A library for interacting with Deepwoken data in a more convenient format, with a few added utilities.

## Usage
The library is written in rust, but bindings for other languages are provided.

### Rust

To use as a dependency:

```
cargo add deepwoken
```

To build:

```
cd rust
cargo build
```

### Typescript

To use as a dependency:

```
npm add deepwoken
```

To build: 

```
cd ts
npm run build
```
